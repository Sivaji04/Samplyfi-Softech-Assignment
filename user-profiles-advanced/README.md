## 📌 Overview

This is **Assignment 2** from the Simplyfi ReactJS Frontend Developer challenge.
A responsive single-page application that fetches user data from a public API and displays it as **interactive profile cards**.

Key concepts demonstrated:

* ReactJS state & props
* Component reusability
* Event handling & conditional rendering
* Responsive layouts with Ant Design
* Dark mode toggle & UI enhancements

---

## ✨ Features

### Core Features

* **Fetch & Display Users** from [JSONPlaceholder](https://jsonplaceholder.typicode.com/users)
* **Avatars** generated dynamically using [DiceBear Avatars API](https://avatars.dicebear.com/)
* **Edit User** via modal form
* **Delete User** dynamically from the list
* **Like/Favorite Users** with visual feedback

### UX Enhancements

* **Search Bar:** Filter users by name, username, or company
* **Sort Users:** Alphabetically by name or company
* **Dark Mode:** Toggle between light and dark themes
* **Loading Spinner:** Shown while fetching API data

### Responsive Design

* Mobile, tablet, and desktop-friendly using Ant Design Grid (`Row` & `Col`)

---

## 🛠️ Tech Stack

* **Frontend:** ReactJS, Ant Design, JavaScript (ES6+)
* **API:** JSONPlaceholder (User Data), DiceBear (Avatars)
* **Deployment:** Vercel

## 💻 Installation

1. Clone the repository:

```bash
git clone <your-github-repo-link>
cd assignment2-advanced
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm start
```

The app will run at [http://localhost:3000](http://localhost:3000)
